공지 바 (Announcement Bar)UI 크롬

구성 / 위치

  • 최상단 고정 (헤더 위)
  • 텍스트 1줄 + 우측 닫기(✕) 버튼
  • 배경: 브랜드 그린(#54BD84), 흰색 텍스트

관리 기능

  • 공지 텍스트 입력 및 수정
  • 노출 기간 설정 (시작일 ~ 종료일)
  • 배경색 / 텍스트색 설정
  • 노출 ON/OFF 토글

동작 방식

  • 닫기 버튼 클릭 시 숨김 (세션 유지)
  • 노출 기간 외 자동 비노출
  • 모바일 / 데스크탑 동일 노출
헤더 (TopAppBar) + 카테고리 GNBUI 크롬

구성

  • 로고 (좌) · 검색창 (중앙) · 장바구니·찜·알림 (우)
  • 스크롤 시 상단 고정 sticky (backdrop-blur)
  • GNB: 가로 스크롤 탭 카테고리 목록 (9개)

관리 기능

  • 로고 이미지 교체
  • 검색창 플레이스홀더 문구 설정
  • GNB 카테고리 생성 / 수정 / 삭제
  • 카테고리 노출 순서 드래그 정렬
  • 카테고리별 연결 URL 및 뱃지(NEW·HOT) 설정

동작 방식

  • 스크롤 시 헤더 sticky 전환 + 그림자 표시
  • GNB 탭 클릭 → 해당 카테고리 페이지 또는 앵커 이동
  • 장바구니 뱃지 숫자 동적 갱신
  • 활성 탭: border-bottom 2px solid #000
히어로 배너 (Rolling Carousel)고정

구성

  • 최대 N개 슬라이드 자동 롤링 (현재 3개)
  • 슬라이드 타입: 이미지형 / 텍스트+배경색형
  • 좌우 화살표 + 하단 도트 인디케이터
  • 5초 자동 전환

관리 기능

  • 슬라이드 추가 / 수정 / 삭제
  • 이미지형: 모바일·데스크탑 이미지 각각 업로드
  • 텍스트형: 배경색(단색·그라디언트) + 헤드라인 입력
  • 버튼 라벨 및 링크 URL 설정 (최대 2개)
  • 자동 전환 간격(초) 설정
  • 슬라이드별 노출 기간 설정

동작 방식

  • setInterval 5000ms 자동 슬라이드 전환
  • 도트 클릭 시 해당 슬라이드로 즉시 이동, 타이머 리셋
  • 좌우 화살표 수동 이동 가능
  • CSS transform: translateX 슬라이드 애니메이션
실시간 타임 특가 & 베스트고정

구성

  • 섹션 타이틀 + 카운트다운 타이머
  • 상품 5개 가로 그리드
  • 카드별: 이미지 · 카테고리 뱃지(green) · 할인율 뱃지(red) · 가격 · 찜 버튼

관리 기능

  • 섹션 타이틀 / 부제 문구 수정
  • 타이머 종료 시각 설정
  • 진열 상품 선택 및 순서 정렬 (최대 5개)
  • 섹션 노출 ON/OFF 토글

동작 방식

  • 카운트다운 타이머 JS setInterval 실시간 갱신
  • 타이머 만료 시 섹션 자동 비노출 또는 다음 특가 전환
  • 상품 호버 시 "장바구니 담기" 슬라이드업 오버레이 노출
MD's Pick (우수 입점 마켓 쇼케이스)고정

구성

  • 3개 쇼케이스 카드 (이미지 배경 + 텍스트 오버레이)
  • 카드별: 배경 이미지 · 태그 뱃지 · 마켓명
  • 목적: MD 추천 상품·컬렉션 기획전 노출

관리 기능

  • 쇼케이스 카드 추가 / 수정 / 삭제
  • 카드별 이미지, 태그, 타이틀, 연결 URL 설정
  • 카드 노출 순서 정렬
  • 섹션 노출 ON/OFF 토글

동작 방식

  • 각 카드 클릭 시 설정 URL로 이동
  • background-image 위 linear-gradient overlay (to top)로 텍스트 가독성 확보
  • 호버 시 scale 또는 border 강조 효과
에디토리얼 / 캠페인 배너고정

구성

  • 배경: #f8f9fa (surface-container-low)
  • 좌: eyebrow(그린) · 헤드라인 · CTA 버튼 2개
  • 우: 캠페인 이미지 (banners/13)
  • 목적: 시즌 캠페인·기획전 강조 노출

관리 기능

  • 배경 이미지 교체 (모바일 / 데스크탑 각각)
  • 텍스트(소제목·헤드라인·설명) 수정
  • 버튼 라벨 및 링크 URL 설정 (최대 2개)
  • 섹션 노출 ON/OFF 토글

동작 방식

  • 배경 이미지 없으면 #f8f9fa 컬러로 폴백
  • 텍스트: 좌측 정렬 (모바일: 세로 스택)
  • CTA 버튼 클릭 시 설정 URL 이동
